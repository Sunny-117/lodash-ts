::: tip Lodash
Lodash 是一个一致性、模块化、高性能的 JavaScript 实用工具库。

Lodash 包含数组的排序算法和过滤、集合的算法、数学函数、与字符串的变更函数，共包含超过一百个函数。Lodash 使用的函数式编程模式允许开发者同时串连多数个函数。

同时，Lodash 的源码分析也可以提升自己的 JavaScript 功底
:::
